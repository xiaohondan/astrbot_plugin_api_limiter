## [2.4.6] - 2026-05-02

### 安全

- **统计面板默认绑定本机**：新增 `stats_bind` 配置项，默认值 `127.0.0.1`（仅本机访问），不再绑定 `0.0.0.0` 暴露公网。用户可手动改为 `0.0.0.0` 开放局域网访问，开启时会显示安全提醒。
- **Token 认证默认强制开启**：`stats_token` 为空时自动生成 32 位随机 token 并保存到配置中，不再允许无 token 访问统计面板。

### 变更

- 开启统计面板时的提示信息不再默认显示局域网 IP，仅在绑定 `0.0.0.0` 时显示
- 仓库地址更新为组织仓库 `NekoAiDev/astrbot_plugin_api_limiter`
- 版本号升级至 2.4.6

## [2.3.0] - 2026-04-30

### 新增

- **黑名单功能**：指定用户直接拒绝，不占用配额，通过 `blacklist` 配置项设置
- **分时段限频**：按小时设置不同限频参数（间隔/次数/冷却），通过 `timeslots` 配置项设置（JSON 格式）
- **超额提醒**：每日配额用到 80% 时自动提醒管理员
- **重置指令**：新增 `重置限频`（全部重置）、`重置冷却`、`重置配额`、`重置对话`、`清空日志` 五个管理员指令
- **导出日志**：管理员发送 `导出日志` 可导出拦截记录为文本
- **群聊独立配额**：不同群设置不同的每日配额，通过 `group_quotas` 配置项设置
- **拦截日志**：所有拦截记录在 WebUI 面板和聊天统计中展示（最多保留 200 条）

### 变更

- 版本号升级至 2.3.0
- WebUI 面板新增黑名单拦截统计、分时段状态、群独立配额数量、拦截记录展示

## [2.2.0] - 2026-04-29

### 新增

- **对话切断功能**：防止短时间内耗尽 API 余额，支持「单用户」和「全局」两种模式（二选一）
  - 单用户模式：每个用户独立计数，达到上限只阻断该用户
  - 全局模式：所有人加起来计数，达到上限全部阻断
  - 私聊和群聊分开计算
  - 新增配置项：`quota_mode`（模式选择）、`quota_limit`（上限条数）、`quota_cooldown_minutes`（冷却时间）
- **对话切断拦截统计**：聊天统计和 WebUI 面板新增对话切断拦截次数显示

### 变更

- 版本号升级至 2.2.0
- 每日重置时同步重置对话切断计数和冷却记录
- 新功能配置项默认为空/0，完全向后兼容

## [2.1.0] - 2026-04-29

### 新增

- **WebUI 统计面板**：管理员发送 `开启统计面板` 开启浏览器统计页面，展示今日调用、累计放行/拦截、放行率、冷却状态进度条、当前配置状态
- **JSON API 接口**：面板提供 `/api` 端点，返回 JSON 格式统计数据
- **统计面板端口配置**：新增 `stats_port` 配置项（默认 6285），可自定义面板端口
- **聊天统计指令**：`/webui` 在聊天中显示统计数据，底部附带开启网页面板的提示
- **`关闭统计面板` 指令**：支持手动关闭统计面板

### 变更

- 版本号升级至 2.1.0
- 插件卸载时自动关闭 WebUI 服务器
- 记录冷却总时长，面板可展示冷却进度条

## [2.0.0] - 2026-04-29

### 新增

- **每日调用配额**：每天最多 API 调用次数（次日自动重置），通过 `daily_limit` 配置项设置
- **白名单模式**：指定不受限频影响的用户 ID，通过 `whitelist` 配置项设置
- **自定义拒绝消息**：拦截时自动回复用户的消息，通过 `reject_message` 配置项设置
- **调用统计面板**：发送 `限频统计` 指令查看今日/累计放行、拦截、冷却触发等数据

### 修复

- 修复 `ProviderRequest` 导入路径错误（`astrbot.api.llm` → `astrbot.api.provider`），解决运行时报错 `No module named 'astrbot.api.llm'`

### 变更

- 仓库迁移至 `astrbot_plugin_api_limiter`
- 版本号升级至 2.0.0
- 所有新功能配置项默认为 0 / 空，完全向后兼容

## [1.0.0] - 2026-04-29

### 新增

- **调用间隔限制**：每次 API 调用前强制等待冷却时间
- **次数限制 + 冷却重开**：达到指定次数后暂停 API，等待后自动恢复
- **安静时段**：指定时间段内完全禁止 API 调用，防止扰民
